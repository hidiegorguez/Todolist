# To-Do List Application

**A practical to-do list application crafted with modern web technologies. Manage your tasks with an intuitive interface and a responsive backend.**

**Note:** The code and comments within the project are in Spanish. This project is currently a work in progress.

## Overview

This project is structured as a full-stack application, combining a front-end built with HTML/CSS/JavaScript and a backend developed with Node.js. It's an effective tool to streamline your daily task management.

## Features

- **Frontend**: Provides a straightforward interface for adding and displaying tasks, featuring functionalities for task creation and removal.
- **Backend**: Utilizes an Express server to handle HTTP requests and serve task data.

## Project Structure

1. **Adjustments**: Sets up the project with necessary installations and libraries.
2. **Class**: Defines the frontend structure with HTML elements and CSS for app styling.
3. **Task Management**: Implements functionalities to interact with the task list, including adding new tasks and removing existing ones.
4. **App Execution**: Includes backend server code with Node.js and Express to manage tasks and serve the task list.

## Getting Started

To get the app running:
1. Clone the repository.
2. Navigate to the project directory.
3. Run `npm install` to install dependencies.
4. Start the server with `node server.js`.
5. Open `index.html` in your browser to use the application.
